# Music Files

Place your music files here following the naming pattern:
- `{ID}-description.extension`
- `{ID}_description.extension`
- `{ID} description.extension`

Examples:
- `0-title-theme.mp3`
- `1_level-bgm.mp3`
- `2 boss-battle.ogg`

Supported formats: `.wav`, `.mp3`, `.ogg`

Music loops continuously. The console extracts the numeric ID and ignores everything after the first non-digit character.
